# Keen Keeper

A modern web application designed to help you maintain and strengthen your relationships by tracking meaningful interactions with friends and loved ones.

## 🛠️ Technologies Used

- **Next.js** - React framework
- **React** - Modern UI library
- **TailwindCSS 4** - Utility-first CSS for styling
- **DaisyUI** - Component library built on Tailwind
- **Recharts** - Charting library for data visualization
- **Lucide React** - React icon library
- **React Toastify** - Toast notifications

## Key Features

### 1. **Friends Dashboard & Profiles**

Comprehensive dashboard of your friends with detailed profiles. View personal information, interaction history, and relationship insights.

### 2. **Quick Check-In System**

Interact via call, text, or video, and log your connections effortlessly. Keep track of when you last engaged with each friend to maintain meaningful, consistent interactions.

### 3. **Interactive Timeline & Statistics**

Visualize your relationship activity with beautiful charts and timeline views. Data-driven approach to maintaining connections.
